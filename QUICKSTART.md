# Loan Management System

## 🚀 Quick Setup

### 1. Installation
```bash
git clone https://github.com/Officialsky54/loan-management-system.git
cd loan-management-system
composer install
npm install
```

### 2. Web Installer
- Visit: `http://your-domain/install`
- Fill in database and admin credentials
- Click "Complete Installation"

### 3. Login
- Admin URL: `http://your-domain/admin`
- Use your admin email and password
